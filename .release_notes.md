# TokiSync v1.9.1 Release Notes

## 🚀 주요 업데이트 및 개선 사항 (v1.8.1 -> v1.9.1)

### 1. UI/UX 현대화 (Glassmorphism)
- **모던 디자인 도입**: 전체 인터페이스를 유리 질감의 투명한 디자인으로 개편하여 시각적 완성도를 높였습니다.
- **탭 기반 레이아웃**: 메뉴 너비를 520px로 확장하고 탭 기반 시스템을 도입하여 정보 접근성을 개선했습니다.

### 2. 소설 다운로드 시스템 강화
- **TXT 포맷 지원**: 기존 EPUB 외에 일반 텍스트(TXT) 포맷으로 소설을 소장할 수 있는 옵션을 추가했습니다.
- **강력한 복호화 파이프라인**: API 응답 내 불필요한 CSS 및 메타데이터를 정밀하게 제거하는 Regex 기반 클리너를 도입했습니다.

### 3. 통신 및 업로드 아키텍처 개선
- **Direct Drive API 연동**: GAS Relay를 거치지 않고 구글 드라이브로 직접 업로드하는 기능을 추가하여 대용량 파일 업로드 속도를 혁신적으로 개선했습니다.
- **Stateless V3 프로토콜**: 서버 사이드 검증 로직 강화에 맞춰 `protocolVersion: 3`과 `folderId` 강제 전송 규약을 적용하여 데이터 무결성을 확보했습니다.

### 4. 다운로더 지능화 및 안정화
- **정렬 로직 수정**: 다운로드 범위 설정 시 1화부터 N화까지 오름차순으로 안정적으로 처리되도록 보완했습니다.
- **실패 리포트 생성**: 네트워크 오류 등으로 다운로드되지 않은 항목을 자동으로 요약하여 `.txt` 리포트로 제공합니다.
- **CORS 우회 기술**: `fetchBlobWithXHR` 도입으로 이미지 호스트의 Referer 차단을 효과적으로 회피합니다.

## 📦 배포 자산 (Assets)
- `tokiSync.user.js`: Tampermonkey 설치용 클라이언트 스크립트
- `TokiSync_Server_Bundle.gs`: GAS 에디터 복사용 서버 통합 번들 소스

---
**설치 방법**: 
1. `tokiSync.user.js`를 클릭하여 Tampermonkey에 설치합니다.
2. 구글 앱 스크립트(GAS) 에디터에 `TokiSync_Server_Bundle.gs` 내용을 전체 복사하여 붙여넣고 배포합니다.

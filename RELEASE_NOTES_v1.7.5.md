# 🚀 TokiSync v1.7.5 Release Notes

> "시청은 몰입감 있게, 다운로드는 멈춤 없이!"

이번 v1.7.5 업데이트는 뷰어의 **네트워크 독립성**과 **백그라운드 효율성**에 집중했습니다. 이제 더 이상 다운로드 때문에 뷰어 화면을 띄워놓고 기다릴 필요가 없습니다.

---

## 🏗️ 1. 다운로드 매니저 (Download Manager) 신설
에듀케이션 센터와 같은 강력한 관리 뷰가 추가되었습니다.
- **실시간 대기열 모니터링**: 현재 어떤 에피소드가 다운로드 중인지, 얼마나 남았는지 직관적으로 확인하세요.
- **개별 제어**: 원하지 않는 다운로드는 목록에서 즉시 제거하거나 취소할 수 있습니다.
- **진입 방법**: 상단 내비게이션 바에 새롭게 추가된 **구름 아이콘(☁️)**을 클릭하세요.

## ⚡ 2. 중단 없는 백그라운드 전송
기존에는 뷰어를 나가면 모든 네트워크 작업이 중단되었지만, 이제는 다릅니다.
- **네트워크 이중화**: 뷰어를 닫거나 다른 에피소드로 이동해도, 매니저에 등록된 다운로드는 백그라운드에서 끝까지 완료됩니다.
- **스마트 프리로드 중단**: 현재 보고 있는 뷰어를 닫으면 불필요한 '다음 화 미리 읽기'는 즉시 멈춰 사용자의 귀중한 데이터를 아껴줍니다.

## 🧹 3. 더 똑똑해진 저장소 관리 (LRU Cache)
기기의 용량을 효율적으로 사용하기 위한 자동 청소 기능이 고도화되었습니다.
- **자동 가비지 컬렉션(GC)**: 최신 읽은 순서대로 캐시를 관리하며, 공간이 필요할 때 오래된 데이터부터 스스로 정리합니다.
- **버그 해결**: 에피소드 수가 적을 때 청소 기능이 멈춰버리던 고질적인 문제를 완벽히 해결했습니다.

## 🛠️ 4. 기타 개선 사항
- **라이브러리 속도 향상**: 데이터베이스 구조 최적화로 에피소드 목록이 더 빠르게 로드됩니다.
- **취소 버튼 추가**: 다운로드 중인 화면에서 바로 작업을 취소하고 목록으로 나갈 수 있는 버튼이 추가되었습니다.

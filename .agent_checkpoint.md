4. **마이그레이션 스크립트 작성**:
   - `Migrate_Service.gs` 내 `Migrate_KavitaFormat` 신규 작성. 파일명 일괄 변경, `info.json` 기반 소스 ID 자가 치유 및 중복 작업 방지용 `kavitaMigrated: true` 완료 플래그 자동 기록 처리.

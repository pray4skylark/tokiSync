# 🚀 TokiSync v1.8.1 릴리스 노트

v1.8.0 정식 릴리스 이후 보고된 핵심 버그들을 수정하고, 소설 뷰어의 완성도를 한 단계 더 높인 **v1.8.1 안정화 업데이트**를 발표합니다.

---

## 🌟 주요 업데이트 사항

### 1. 🛡️ GAS V8 런타임 호이스팅 버그 완벽 해결
- **문제**: Google Apps Script V8 엔진에서 최상위 `const` 변수들이 파일 로딩 순서에 따라 `ReferenceError`를 일으키던 고질적인 문제를 해결했습니다.
- **해결**: 모든 전역 서비스(`DriveAccessService`, `Debug` 등)의 선언을 호이스팅이 보장되는 `var` 키워드로 변경하여 뷰어 구동 안정성을 100% 확보했습니다.

### 2. 📖 V2 소설 뷰어 고도화 (Precision Reading)
- **툴바 완벽 이식**: V1의 핵심 설정 기능(테마, 폰트 크기, 줄 간격)을 V2 플로팅 툴바에 완벽히 통합했습니다.
- **Locator 엔진 도입**: 단순 비율 기반 동기화를 폐기하고 DOM Offset 기반의 정밀 계산을 도입했습니다. 이제 폰트나 레이아웃이 변해도 읽던 문단을 1px의 오차 없이 유지합니다.
- **가독성 최적화**: 본문 최대 폭을 소설책 표준 규격인 **720px**로 고정하여 몰입감을 높였습니다.

### 3. ⚡ 다운로드 안정성 및 유연성 강화
- **속도 제어 확장**: 봇 탐지 우회를 위해 `느림(5-15초)` 및 `매우 느림(10-30초)` 정책을 추가했습니다.
- **API 복호화 보호**: 소설 API 복호화 시 서버 부하 및 IP 차단을 방지하기 위해 가장 안전한 속도 정책을 자동으로 강제 적용합니다.
- **정렬 로직 개선**: 특정 범위 다운로드 시 회차 번호 순서가 꼬이던 현상을 수정하여 정방향(1화부터) 순차 다운로드를 보장합니다.

---

## 🛠️ 기술적 개선 사항

- **진도 유실 방지**: 에피소드 종료 시 즉시 DB에 위치를 저장하는 Flush 로직 강화.
- **데이터 오염 차단**: 새 에피소드 진입 시 이전 마커 정보를 확실히 초기화하는 Reset 프로세스 도입.
- **빌드 최적화**: UserScript 메타데이터 및 가이드 문서의 모든 버전을 v1.8.1로 통일.

# 🏁 TokiSync v1.26.4 Deployment Checklist

배포 전 최종 안정성을 보장하기 위한 체크리스트입니다. v1.26.4의 전면 개편된 UI와 소설 엔진 픽스 사항들이 모두 반영되었는지 확인하세요.

---

## 1. 빌드 및 버전 관리 (Build & Versioning)
- [ ] `package.json` 버전이 `1.26.4`으로 명시되어 있는가?
- [ ] `webpack.core.config.cjs`의 UserScript 버전이 `1.26.4`으로 갱신되었는가?
- [ ] `npm run build` 실행 시 에러 없이 모든 어셋(Viewer, Core, GAS Bundle)이 생성되는가?
- [ ] `docs/tokiSync.user.js` 파일이 정상적으로 빌드되었는가?

## 2. UI/UX 정합성 (Modernized UI)
- [ ] 모달 너비가 **520px**로 확장되어 가독성이 충분한가?
- [ ] **탭 시스템**(다운로드, 설정, 시스템) 간의 전환이 부드럽고 정상적으로 작동하는가?
- [ ] 각 탭의 버튼 및 설정 항목들이 의도한 레이아웃대로 배치되어 있는가?
- [ ] 다크/라이트 모드 및 글래스모피즘 효과가 UI 전반에 잘 적용되었는가?

## 3. 소설 엔진 및 다운로드 (Novel & Downloader)
- [ ] 소설 API 복호화 시 **사용자가 설정한 속도 정책**이 올바르게 반영되는가? (강제 제한 해제 확인)
- [ ] 소설 출력 포맷(**EPUB / TXT**) 선택이 실제 다운로드 파일에 정상 반영되는가?
- [ ] 복호화된 텍스트에서 불필요한 JSON 데이터나 이스케이프 문자가 제거되었는가?
- [ ] 다운로드 실패 시 **실패 리포트(.txt)** 파일이 자동으로 생성되는가?

## 4. 안정성 및 동기화 (Stability & Sync)
- [ ] GAS URL 설정이 없을 때 `fetchHistory` 등 네트워크 요청이 사전에 차단되는가?
- [ ] 5단계 다운로드 속도 정책이 누락 없이 작동하는가?
- [ ] 범위 다운로드 시 회차 번호 오름차순 정렬이 보장되는가?
- [ ] 백그라운드 다운로드 중 안티-슬립(오디오 재생)이 활성화되는가?

## 5. 최종 배포 및 공지 (Release Procedure)
- [ ] `CHANGELOG.md` 및 `RELEASE_NOTES_v1.26.4.md`가 최신화되었는가?
- [ ] `gh-pages` 브랜치에 최신 빌드 결과물이 푸시되었는가?
- [ ] 실사용 환경에서 통합 메뉴(`Ctrl+Shift+T`)가 정상 호출되는가?
